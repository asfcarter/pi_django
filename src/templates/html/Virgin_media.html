Virgin Media Change

Porting the setting from BT to Virgin Media was not easy the following was required:

IP Address - BT uses 198.162.1.* , Virgin uses 198.162.0.* therefore the static address was requried to change.

Port Forwarding - The BT hub uses default settings for a web server (which I can assume to be port 80). For Virgin the port settings for port forwarding are explicit. Both the internal and external ports need to be set to 80 for the web server.

DNZ - I'm not sure whether this is necessary, but I saw it on some forums that Virgin Hub 3 had issues which required the DNZ to be set on the specifically used IP address to allow traffic.

Websafe - Virgin has a number of Apps on their registration form to make you safe. These also block incoming services. Hence turning off websafe.

A useful application was:
http://www.canyouseeme.org/
I set my laptop to be port forwarded on port 80 and ran that website and it worked perfectly.
