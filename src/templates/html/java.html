Java

In order to run smartgit/hg I had to upgrade from java 1.5 to 1.7. Note the 5 and 7 are the large numbers I.e.
1.5 -> java5
1.7 -> java7

In order to check the current version of java you can run the following command:
    java -version

In order to install java on Ubuntu you run the following command:
    sudo apt-get install openjdk-7-jdk
