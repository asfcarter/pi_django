I installed the following two components (not certain that the first is required):
  •     sudo apt-get install apache2-bin
  •     sudo apt-get install apache2
Initially under apache there was only one www directory where the html files were stored. In recent releases the html files now have their own subdirectory, the reason being for security, therefore this has been recently changed.

Symbolic links are now used to link the git repository to the /var/www directory. This is done via a similar command to:
  • ln -s /home/asfcarter/cricket/www /var/www
          (Note: the path where the complete path to the git repository needs to be executable to other users)

Initially when I used the new html directory I kept the old linking style to www, this was done by modifying the available 000 and another 000 file within /etc/apache/.....

Restart apache:
sudo /etc/init.d/apache2 restart
